# Claude Public Skills (Built-in)

These are Anthropic's official skills that come pre-installed with Claude.

## Skills Included

| Skill | Description | Size |
|-------|-------------|------|
| `docx` | Word document creation, editing, tracked changes, comments | 1.2MB |
| `xlsx` | Excel spreadsheet creation, formulas, formatting, analysis | 21KB |
| `pptx` | PowerPoint presentation creation and editing | 1.3MB |
| `pdf` | PDF manipulation, text extraction, form filling, merging | 57KB |
| `frontend-design` | Production-grade UI/frontend interface creation | 15KB |
| `product-self-knowledge` | Anthropic product reference for accurate answers | 2.5KB |

## Structure

Each skill contains:
- `SKILL.md` - Main instructions
- `scripts/` - Supporting code (where applicable)
- `LICENSE.txt` - License information
- Additional reference files

## Usage

These skills are automatically available in Claude. Reference them in your own skills or study them to understand Anthropic's skill architecture patterns.
