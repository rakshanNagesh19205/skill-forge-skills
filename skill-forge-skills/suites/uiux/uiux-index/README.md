# UI/UX Design Suite — Index

## Quick Start

This is the **hub skill** for the UI/UX design suite. It routes you to the right specialized skill based on your task.

## Available Skills

| Skill | Use For |
|-------|---------|
| `uiux-systems` | Design tokens, foundations, from-scratch setup |
| `uiux-visual` | Swiss design, typography, color, visual polish |
| `uiux-interaction` | React Bits components, animations, motion |
| `uiux-usability` | UX audits, fixing existing interfaces |
| `uiux-a11y` | Accessibility, WCAG compliance |

## Usage

Simply describe what you're trying to do, and this skill will route you to the right place:

```
"I need to build a dashboard from scratch"
→ Routes to uiux-systems

"Make my landing page look better"
→ Routes to uiux-visual

"Add some cool animations"
→ Routes to uiux-interaction
```

## Defaults

- **Design Philosophy:** Swiss (International Typographic Style)
- **Component Library:** React Bits (reactbits.dev)
- **Framework:** React with TypeScript + Tailwind
