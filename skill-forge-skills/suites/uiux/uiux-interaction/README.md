# UI/UX Interaction Design — React Bits & Motion

## Quick Start

This skill adds purposeful animations and interactive components using React Bits (reactbits.dev).

## When to Use

- Adding animations to headlines/text
- Implementing hover/focus interactions
- Creating animated UI components
- Adding background effects
- Building micro-interactions

## React Bits Library

110+ animated components:
- **Text:** SplitText, BlurText, RotatingText, CountUp
- **UI:** AnimatedList, TiltCard, Dock, Accordion
- **Backgrounds:** Particles, Aurora, GridPattern
- **Effects:** Spotlight, Ripple, Trail

## Installation

```bash
# shadcn CLI (recommended)
npx shadcn@latest add @react-bits/SplitText-TS-TW

# jsrepo CLI
npx jsrepo add https://reactbits.dev/ts/tailwind/TextAnimations/SplitText
```

## Swiss Motion Philosophy

- ✅ Purposeful motion (feedback, orientation, hierarchy)
- ❌ Decorative animation (bouncing, wiggling, gratuitous effects)

## Quick Triggers

```
animate     → Add purposeful animation
reactbits   → Browse component options
text-fx     → Text animation options
background  → Background effects
hover       → Hover interaction patterns
```
