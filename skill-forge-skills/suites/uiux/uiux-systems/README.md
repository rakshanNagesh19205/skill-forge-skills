# UI/UX Design Systems

## Quick Start

This skill creates design system foundations: tokens, spacing scales, typography systems, color palettes, and component architectures.

## When to Use

- Starting a new project from scratch
- Setting up design tokens
- Creating consistent spacing/type scales
- Building theming architecture

## What It Produces

- CSS Custom Properties
- Tailwind configuration
- TypeScript token objects
- Semantic token layers

## Swiss Design Defaults

- **Spacing:** 4px base unit
- **Typography:** 1.25 scale ratio, sans-serif only
- **Colors:** Restrained palette (neutral + 1 accent)
- **Grid:** 12-column modular grid

## Quick Triggers

```
tokens      → Complete token system
spacing     → Spacing scale only
typography  → Type system only
colors      → Color palette only
theme       → Theming architecture
```

## After This Skill

Once foundations are set, move to:
- `uiux-visual` → Apply Swiss design principles
- `uiux-interaction` → Add React Bits components
