# UI/UX Visual Design — Swiss Style

## Quick Start

This skill applies Swiss/International Typographic Style principles to create clean, professional interfaces.

## When to Use

- Making an interface look better
- Applying brand styling
- Setting up typography hierarchy
- Creating Swiss grid layouts
- Cleaning up "AI slop" aesthetics

## Swiss Principles Applied

1. **Clarity Over Cleverness** — Communication first
2. **Mathematical Order** — Grids, scales, ratios
3. **Objective Presentation** — Information over decoration
4. **Asymmetric Balance** — Dynamic tension
5. **Purposeful Restraint** — Limited palette, maximum impact

## Quick Triggers

```
swiss       → Full Swiss design treatment
grid        → Set up grid layout
type        → Apply typography hierarchy
color       → Apply color principles
whitespace  → Optimize spacing
```

## Anti-Patterns Avoided

- ❌ Decorative gradients
- ❌ Excessive shadows
- ❌ Centered body text
- ❌ Multiple competing fonts
- ❌ Pill-shaped everything
- ❌ Decorative animations

## After This Skill

- `uiux-interaction` → Add purposeful animations
- `uiux-a11y` → Verify accessibility
